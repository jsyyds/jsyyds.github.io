居中布局

布局解决方案——居中布局

文章最后更新于：2021年5月11日

  • 水平居中
  • inline-block + text-align
  • table + margin
  • absolute + transform
  • flex + justify-content
  • 垂直居中
  • table-cell + vertical-align
  • absolute + transform
  • flex + align-items
  • 水平垂直居中
  • inline-block + text-align + table-cell + vertical-align
  • absolute + transform
  • flex + justify-content + align-items

水平居中

inline-block + text-align

<div class="parent">
    <div class="child">DEMO</div>
</div>
DEMO
.child {
    display: inline-block;
}
DEMO
.child {
    display: inline-block;
}
.parent {
    text-align: center;
}
DEMO

优点：

  • 兼容性非常好

    IE6、7 不支持 inline-block，但可以通过 dispaly:inline;zoom:1; 模拟 inlien-block，所以这种方法可以兼容到 IE6、7 以上的浏览器。

缺点：

  • text-align 被子元素继承

    child 元素会继承 parent 元素的 text-align 设置，所以 child 的样式中要补上一条 text-align: left; 来还原。

table + margin

<div class="parent">
    <div class="child">DEMO</div>
</div>
DEMO
.child {
    display: table;
}
DEMO
.child {
    display: table;
    margin: 0 auto;
}
DEMO

优点：

  • 只设置了 child 的样式，不关心 parent 上的样式

  • 兼容性 IE8+

    想要兼容 IE6、7，可以换成 <table> 结构。

absolute + transform

<div class="parent">
    <div class="child">DEMO</div>
</div>
DEMO
.parent {
    position: relative;
}
.child {
    position: absolute;
}
DEMO
.parent {
    position: relative;
}
.child {
    position: absolute;
    left: 50%;
}
DEMO
.parent {
    position: relative;
}
.child {
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
}
DEMO

优点：

  • absolute 定位的元素是脱离文档流的，不会对其他元素造成影响

缺点：

  • 不支持 IE8 及以下浏览器

    高版本浏览器 transform 可能需要加一些私有前缀

flex + justify-content

<div class="parent">
    <div class="child">DEMO</div>
</div>
DEMO
.parent {
    display: flex;
}
DEMO
/* 方法一 */
.parent {
    display: flex;
    justify-content: center;
}

/* 方法二 */
.parent {
    display: flex;
}
.child {
    margin: 0 auto;
}
DEMO

优点：

  • 只需要设置父元素（方法一）

缺点：

  • 不支持 IE9 及以下浏览器

</div>
DEMO

inline-block + text-align + table-cell + vertical-align

.parent {
    text-align: center;
    display: table-cell;
    vertical-align: middle;
}
.child {
    dispaly: inline-block;
}
DEMO

优点：

  • 兼容性非常好

    IE6、7 不支持 inline-block，但可以通过 dispaly:inline;zoom:1; 模拟 inlien-block，所以这种方法可以兼容到 IE6、7 以上的浏览器。

    换为 <table> 结构可以兼容 IE6、7

absolute + transform

.parent {
    position: relative;
}
.child {
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
}
DEMO

优点：

  • 脱离文档流，子元素不会干扰其他元素

缺点：

  • 不支持 IE8 及以下浏览器

    高版本浏览器 transform 可能需要加一些私有前缀

flex + justify-content + align-items

.parent {
    display: flex;

    align-items: center;
}
DEMO

优点：

  • 只需要设置父元素

缺点：

  • 不支持 IE9 及以下浏览器
